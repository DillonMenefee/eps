eps

Embedded PowerShell (similar to ERB)

It's crazy to create templating for PowerShell

test.eps:

Hi <%= $name %>

Please buy me the following items:
<% 1..5 | %{ %>
  - <%= $_ %> pigs ...
<% } %>

Dave is a <% if($true) { %> boy <% } else { %> girl <% } %>. 

Thanks,
Dave
<%= (Get-Date -f yyyy-MM-dd) %>

commands:

$text = gc .\test.eps
$text = $text -join "`n"
$name = "ABC"
$script = Compile-Raw $text
iex $script

It will produce:

Hi ABC

Please buy me the following items:
  - 1 pigs ...
  - 2 pigs ...
  - 3 pigs ...
  - 4 pigs ...
  - 5 pigs ...

Dave is a boy.

Thanks,
Dave
